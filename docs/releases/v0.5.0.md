# CoreTemplateStd v0.5.0

This release makes the optional Qt adapter available to installed package consumers.

## Highlights

- `CoreTemplateStd::QtAdapter` is now installed when `CORETEMPLATE_BUILD_QT_ADAPTER=ON`.
- Consumers can request the adapter with `find_package(CoreTemplateStd REQUIRED COMPONENTS QtAdapter)`.
- The package config keeps the default core package Qt-free; Qt is searched only when the `QtAdapter` component is requested.
- CI now verifies the installed Qt adapter with a package smoke consumer.

## Example

```cmake
find_package(CoreTemplateStd REQUIRED COMPONENTS QtAdapter)
target_link_libraries(your_qt_target PRIVATE CoreTemplateStd::QtAdapter)
```
