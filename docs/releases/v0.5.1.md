# CoreTemplateStd v0.5.1

This patch release improves the Qt GUI example controls for canceling tasks by ID.

## Fixes

- Selecting an active task now fills the task ID/type/group fields.
- Newly started tasks are selected automatically, so `Cancel By ID` targets the latest started task by default.
- `Cancel By ID` now logs a clear message when the entered ID is not active.

## Note

Task ID and task type are different values. For non-cooperative tasks, canceling requests stop and reports timeout if the task ignores `stopTaskFlag()`.
